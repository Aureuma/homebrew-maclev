# homebrew-maclev

Homebrew tap for `maclev`.

## Install

```bash
brew tap Aureuma/maclev git@github.com:Aureuma/homebrew-maclev.git
brew install maclev
```

## Upgrade

```bash
brew update
brew upgrade maclev
```

## Formula source

The formula installs `maclev` from the tagged source in:

- `https://github.com/Aureuma/maclev`
